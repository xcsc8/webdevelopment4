wat er fout is aan de code Een < ol> mag niet direct als kind van < ul>. Alleen < li>-elementen mogen direct binnen < ul> staan. De geneste < ol> zit buiten een < li>, maar hoort binnen een < li> te staan. In HTML moet het dus zo zijn: een geneste lijst zit altijd binnen een < li>.
